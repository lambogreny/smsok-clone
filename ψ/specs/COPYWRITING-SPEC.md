## 5. LANGUAGE RULES

### ใช้ภาษาไทย
- หัวข้อ, คำอธิบาย, ปุ่ม, label, placeholder ทั้งหมด
- ข้อความ error / success / loading

### ใช้ English ได้
- คำเทคนิค: SMS, API, OTP, CSV, RESTful, SDK, Node.js, Python, PHP
- ชื่อเฉพาะ: SMSOK, EasySlip, PromptPay
- Format: email, URL, ตัวเลข
- ชื่อแพ็กเกจ (A, B, C...)

### ห้ามทำ
- Mixed EN/TH ในประโยคเดียวโดยไม่จำเป็น เช่น ~~"Send SMS ด่วน"~~ → "ส่งข้อความด่วน"
- ใช้ English เป็น label เมื่อมีคำไทย เช่น ~~"Recipients"~~ → "หมายเลขผู้รับ"
- ใช้ทับศัพท์มากเกินไป เช่น ~~"แคนเซิล"~~ → "ยกเลิก"
