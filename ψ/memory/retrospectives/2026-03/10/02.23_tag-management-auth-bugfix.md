## Lessons Learned

- When TS `as const` is used on an array of objects with literal-typed fields, `useState(array[0].field)` infers the literal type, not `string`. Always use `useState<string>(...)` explicitly for string states derived from const arrays.
- `Promise.all([getContacts(), getTags()])` pattern is cleaner than sequential awaits in Server Components — parallel fetch, lower latency.
- Reaper measures git heartbeat, not conversational presence. During PM loop-check phases, git is silent. Need to find work to commit even during review-blocked periods, or explicitly signal "holding pattern, waiting on reviewer."
