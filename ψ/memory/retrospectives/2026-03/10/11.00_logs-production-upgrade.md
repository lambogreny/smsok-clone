## Lessons Learned
1. When adding a column to a data table, track ALL grid template definitions — header, row, and any responsive variants
2. cURL reconstruction from request logs is a high-value developer feature that's cheap to implement
3. Source (WEB/API) distinction helps ops teams quickly identify whether issues come from the dashboard or customer API integrations
